Com es guarden les dades?
El codi de l'app està allotjat a GitHub i és igual per a tothom. Les teves dades (entrenaments, exercicis, plantilles, objectius) es guarden només al teu dispositiu (localStorage del navegador) — mai s'envien a cap servidor ni són accessibles per a ningú més, només per a tu.

El navegador podria esborrar les teves dades (per exemple, si esborres freqüentment les dades de navegació, reinstal·les el sistema o canvies de dispositiu sense importar una còpia de seguretat...). Recomanem fer una còpia de seguretat (Exportar JSON) de tant en tant des d'aquí mateix. Amb la còpia, sempre podràs recuperar totes les dades.

1RM Epley: pes × (1 + reps/30)
Implícit: pes ÷ (% / 100)
